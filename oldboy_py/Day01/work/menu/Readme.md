#### Day01 - 三级菜单

1. 本三级菜单为经济区选择，包含“京津冀、长三角、珠三角”的部分省市地区。（因时间原因没有写全，每个经济区只选择了三个“省、市”，每个“省/市”只选择了五个“市/区”）
2. 当打开程序后，可根据提示输入相应的数字，来进入下一级的菜单。
3. 在任意一级菜单，按“q”键为退出程序；在二、三级菜单，按“b”为返回上一级菜单。
4. 当输入过大的数字或其他字符时，会有相应提示，请按照提示输入正确的格式。
